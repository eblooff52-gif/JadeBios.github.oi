# Jade Bio

Сайт био-ссылок (профиль, VIP, админка). Работает **без Node.js** — только браузер.

## Файлы (всё в корне)

| Файл | Назначение |
|------|------------|
| `index.html` | Главная страница |
| `style.css` | Стили |
| `app.js` | Логика, localStorage |
| `effects.js` | Снег / листья |
| `default-avatar.svg` | Аватар по умолчанию |
| `README.md` | Инструкция |
| `.gitignore` | Для Git (не загружать лишнее) |

**Итого 7 файлов сайта** — всё в одной папке, без подпапок.

## Как открыть

Дважды щёлкни **`index.html`** или залей папку на **GitHub Pages**.

Профиль: `index.html?u=ник`

## Первый вход

- Админ: `admin` / `admin123`
- Инвайт: `VL-START2025`

## Discord (статус и игра)

1. В Discord включи режим разработчика → скопируй свой **ID**
2. Зайди на https://discord.gg/lanyard → боту `.help`
3. В кабинете вставь ID в поле **Discord ID**

## GitHub

1. Один раз запусти **`УДАЛИТЬ-ЛИШНЕЕ.bat`** (уберёт `node_modules`, если остался).
2. Загружай **только файлы из корня** (см. таблицу выше + `README.md` + `.gitignore`).

В репозитории: Settings → Pages → Source: **main** / root → Save.

Сайт: `https://ТВОЙ_НИК.github.io/ИМЯ_РЕПО/`
